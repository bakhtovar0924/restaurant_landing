## ✨ Что есть на сайте

- Hero — полноэкранный баннер с анимированным фоном и кнопкой бронирования
- О ресторане — история, философия кухни, счётчики (гости, награды, лет опыта)
- Меню — категории блюд с ценами и описаниями
- Галерея — сетка фотографий интерьера и блюд
- Отзывы — карусель с реальными отзывами гостей
- Бронирование — форма с выбором даты, времени и количества гостей
- FAQ — аккордеон с частыми вопросами
- Контакты — карта, телефон, email, соцсети
- Футер — навигация, режим работы, юридические ссылки

## 🚀 Запуск проекта

# Клонировать репозиторий
git clone https://github.com/bakhtovar0924/restaurant_landing.git

# Перейти в папку
cd restaurant_landing

# Установить зависимости
npm install

# Запустить в режиме разработки
npm run dev

# Собрать для продакшена
npm run build
